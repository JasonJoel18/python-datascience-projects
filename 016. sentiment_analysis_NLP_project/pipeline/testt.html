def lemmatize_text(text):
    lemmatizer = WordNetLemmatizer()
    lemmatized_tokens = []
    for word in nltk.word_tokenize(text):
        lemma = lemmatizer.lemmatize(word, pos='v')
        lemmatized_tokens.append(lemma)
    return ' '.join(lemmatized_tokens)

Conclusion

In conclusion, the main aim of this pipeline is to classify the tweets into two categories such as suicidal and non-suicidal. The essential step for this classification includes importing the given dataset, exploring, experimenting with various model, training and hypertuning the models and eventually determining the best model.
The target label for the given dataset is 'suicide' which have values 0 and 1 where 0 represents 'non-suicidal' and 1 represents 'suicidal'.
While doing EDA many unwanted data were discovered. Hence, preprocessing the unwanted data was cleaned.
For stemming the words WordNet Lemmatizer was used due to it's good accuracy. Further, TF-IDF matrix was created and concatenated with the dataframe.
Finally, the data was split into train, validation and test.
Due to heavy imbalance in the data set, smote and adasyn techniques were used to oversample the data.

Next, for training, 4 models were selected for task, such as

  1. Logistic Regression
  2. SVM
  3. Random Forest Classifier
  4. XGBClassifier

All three sets (imbalanced, smote oversampled, adasyn oversamples) of dataset was run through these model to get the best performing mode.
We found that XGBClassifier with SMOTE oversampling data performed the best with G-Mean of 0.864 and F1 score of 0.832
This model is then tuned using Optuna Hypertuning.
This gives us the best performing hyperparameters as 'n_estimators'= 592, 'max_depth'= 6, 'min_child_weight'= 3, 'learning_rate' 0.03786534038927119}

We use these parameters to train our final model. And we got the best result out of all the experiments we conduction with the results below
